Android 3.0 (HoneyComb) en approche !

Catégorie : Language | 01 février 2011 | Tags :  Android, Java

La version d’Android 3.0, nom de code HoneyComb, est disponible depuis une semaine en pre-release. La bonne nouvelle nous vient directement du blog d’android-developers. Après les 3 releases mineures Éclair, Froyo et Gingerbread, nous allons enfin pouvoir tester la prochaine mouture majeure d’Android.

Vous pouvez récupérer ce nouvel Android 3.0 Preview SDK à cette adresse. Gardez bien en tête que ce n’est qu’une pre-release : vous pouvez donc tester les nouvelles API en sachant qu’elles pourront évoluer (méthodes, performance…) entre cette version et la version finale.

Petit tour d’horizon des nouveautés…

… et le moins que l’on puisse dire c’est qu’elles sont nombreuses ! Voici d’ailleurs un petit teaser vidéo :

Tablette First !

Premier point : HoneyComb a été spécialement conçue pour les terminaux à grand écran et en particulier les tablettes. C’est d’ailleurs ce support des grands écrans qui est tout de suite mis en avant dans les highlights. Quelques images :

Nouvelle Interface Utilisateur

Cette nouvelle UI pour tablette apporte entre autres :

  • des widgets et des notifications plus riches
  • des nouveaux thèmes
  • une barre système persistante pour des notifications globales
  • une barre d’action pour l’application en cours
  • un écran d’accueil customisable
  • un menu “applications récentes” dans la barre système tirant ainsi avantage du multitâche (partie centrale de ce nouvel Android) pour passer rapidement d’une application à une autre
  • un drag’n drop amélioré pour l’éditeur de texte
  • un clavier reconçu et adapté (pour tablette donc)
  • une amélioration de la sélection de texte ainsi que du copier/coller
  • des nouvelles options de connectivité (média, photo, USB, mass-storage…)
  • et des améliorations générales d’applications existantes : navigateur, appareil photo, gallerie, contacts, email

Nouvelles Fonctionnalités

Les développeurs ne sont pas en reste et seront ravis de voir plusieurs nouvelles fonctionnalités mises à leur disposition :

  • nouveaux composants graphiques (widgets, notifications, nouvelles barres…)
  • graphismes 2D et 3D : nouvelle API d’animation, accélération hardware pour la 2D et Renderscript, un moteur 3D
  • support du multicore
  • multimédia avec du live streaming par HTTP, DRM…
  • fonctionnalités dites d’entreprise avec du stockage crypté, expiration de mot de passe, complexité du mot de passe…

Mais aussi

A noter aussi quelques mises à jour concernant les outils suivants :

  • SDK Tools (r9)
  • NDK (r5b)
  • ADT Plugin pour Eclipse (9.0.0)

Et la version finale ?

Concernant la version finale, pas de date précise.
Seule information : celle-ci est prévue dans les prochains mois, ce qui ne nous aide pas
En tout cas, nous pouvons déjà tester ce nouvel opus. Je vais d’ailleurs de suite m’attaquer à cette nouvelle API d’animation !
